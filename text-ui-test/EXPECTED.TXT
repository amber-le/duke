   ╚════ ▣ Here are the list of commands: ▣ ════╝
   1.  list - list all the tasks
       ╰┈➤e.g. list
   2.  <task description> - add a todo task
       ╰┈➤e.g. read book
   3.  <task description> /by <dd/mm/yyyy hhmm> - add a deadline task
       ╰┈➤e.g. TIC2002 quiz 8 /by 2/12/2023 1800
       Note: you can type 'dates' to see useful keywords that help you enter dates faster
   4.  event <task description> /from <dd/mm/yyyy hhmm> /to <dd/mm/yyy hhmm> - add an event task
       ╰┈➤e.g. project meeting /from 02/12/2023 1800 /to 02/12/2023 2000
       Note: you can type 'dates' to see useful keywords that help you enter dates faster
   5.  mark <task number> - mark a task as DONE
       ╰┈➤e.g. mark 1
       Note: mark all -> CrabY will mark ALL the tasks as DONE
   6.  unmark <task number> - mark a task as NOT DONE YET
       ╰┈➤e.g. unmark 1
       Note: unmark all -> CrabY will mark ALL the tasks as NOT DONE YET
   7.  delete <task number> - delete a task
       ╰┈➤e.g. delete 1
       Note: delete all -> CrabY will delete ALL the tasks
   8.  find <keyword> - find a task with the keyword
       ╰┈➤e.g. find book
   9.  sort <type/type-r/t/tr/date/d> - sort the list by type or date
       Note: type-r or tr will sort the list by type in revert order
       ╰┈➤e.g. sort t
   10. undo - undo your previous command (except: list, find, help, blah commands)
       ╰┈➤e.g. undo
   11. switch - switch to another list
       ╰┈➤after that CrabY will ask you to enter the name of the list that you want to switch to
   12. help - show the list of commands
       ╰┈➤e.g. help
   13. bye - exit the program
       ╰┈➤e.g. bye

   *** NOTE: if you use the same [List-name] that used before
   ╰┈➤you can continue to use the previous list of tasks.
